## How to set up Beyond the Dream with the Star Rod debug menu

(Please note that this tutorial will assume you have mapped your d-pad, L, and R buttons. If you are testing on console, you do not need to worry about this)
1. Press d-pad left. This will open the Star Rod debug menu for you. 
2. Using d-pad up or down, scroll all the way to "Set Story Byte." This basically changes what point in the story you are.
3. For this tutorial, we'll only be adding Goombario. In this case, set the story byte to "9C" and press R to select. Now the game thinks that you just made it to Merlon back in Toad Town, and you spoke to him about the "Dark Toads," thereby revealing the Koopa Bros.
